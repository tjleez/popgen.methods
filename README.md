# popgen.methods

## SNP matrix
### vcf2fa.py
print multiple sequence fasta from vcf
```
python vcf2fa.py *input.vcf*
```
outputs *input.vcf.fa*

### pairdiv_vcf.fa.py
count pairwise differences between isolates from input.vcf.fa
```
python pairdiv_vcf.fa.py *input.vcf.fa* *output.txt*
```
